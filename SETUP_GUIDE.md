# OptiroRoute Setup Guide\n\n## Step-by-Step Installation\n\n### 1. System Requirements\n\n**Minimum Requirements:**\n- Python 3.8 or higher\n- Node.js 16 or higher\n- 4GB RAM\n- 1GB free disk space\n- Internet connection (for API calls)\n\n**Recommended:**\n- Python 3.10+\n- Node.js 18+\n- 8GB RAM\n- SSD storage\n\n### 2. Check Your System\n\n```bash\n# Check Python version\npython --version\n# Should show: Python 3.8.x or higher\n\n# Check Node.js version\nnode --version\n# Should show: v16.x.x or higher\n\n# Check npm version\nnpm --version\n# Should show: 8.x.x or higher\n```\n\n### 3. Get API Keys (All Free)\n\n#### OpenRouteService (Required)\n1. Go to [openrouteservice.org](https://openrouteservice.org/dev/#/signup)\n2. Click \"Sign Up\" â†’ Create account\n3. Go to Dashboard â†’ \"Request a Token\"\n4. Copy the JWT token (starts with \"eyJ...\")\n\n#### OpenWeather (Required)\n1. Go to [openweathermap.org](https://openweathermap.org/api)\n2. Click \"Sign Up\" â†’ Create account\n3. Go to API Keys â†’ Copy your key\n4. Wait 10 minutes for activation\n\n#### Groq AI (Optional but Recommended)\n1. Go to [console.groq.com](https://console.groq.com)\n2. Sign up with Google/GitHub\n3. Go to API Keys â†’ Create new key\n4. Copy the key (starts with \"gsk_...\")\n\n### 4. Installation Commands\n\n```bash\n# 1. Clone repository\ngit clone <your-repo-url>\ncd optiroute-main\n\n# 2. Create Python virtual environment (recommended)\npython -m venv venv\n\n# 3. Activate virtual environment\n# Windows:\nvenv\\Scripts\\activate\n# macOS/Linux:\nsource venv/bin/activate\n\n# 4. Install Python dependencies\npip install -r requirements.txt\n\n# 5. Setup environment file\ncp .env.example .env\n# Edit .env with your API keys\n\n# 6. Install frontend dependencies\ncd react-frontend\nnpm install\ncd ..\n```\n\n### 5. Configuration\n\nEdit `.env` file with your API keys:\n\n```bash\n# Required\nOPENROUTE_API_KEY=\"your-jwt-token-here\"\nOPENWEATHER_API_KEY=\"your-weather-key-here\"\n\n# Optional (improves chat parsing)\nGROQ_API_KEY=\"your-groq-key-here\"\n\n# Security\nJWT_SECRET_KEY=\"change-this-to-something-secure\"\n```\n\n### 6. Start the System\n\n**Terminal 1 - Backend:**\n```bash\npython final_backend.py\n```\n\n**Terminal 2 - Frontend:**\n```bash\ncd react-frontend\nnpm start\n```\n\n### 7. Verify Installation\n\n1. **Backend Test:**\n   - Open: http://localhost:5001/api/test\n   - Should show: `{\"status\": \"working\"}`\n\n2. **Frontend Test:**\n   - Open: http://localhost:3000\n   - Should show OptiroRoute interface\n\n3. **Full System Test:**\n   - Go to Chat Order tab\n   - Type: \"send from empire state to times square\"\n   - Should show order preview\n\n## Troubleshooting\n\n### Python Issues\n\n**\"python: command not found\"**\n```bash\n# Try python3 instead\npython3 --version\npython3 final_backend.py\n```\n\n**\"ModuleNotFoundError\"**\n```bash\n# Reinstall dependencies\npip install --upgrade pip\npip install -r requirements.txt\n```\n\n**\"Permission denied\"**\n```bash\n# Use virtual environment\npython -m venv venv\nvenv\\Scripts\\activate  # Windows\nsource venv/bin/activate  # macOS/Linux\npip install -r requirements.txt\n```\n\n### Node.js Issues\n\n**\"npm: command not found\"**\n- Install Node.js from [nodejs.org](https://nodejs.org)\n- Restart terminal after installation\n\n**\"EACCES permission denied\"**\n```bash\n# Fix npm permissions\nnpm config set prefix ~/.npm-global\nexport PATH=~/.npm-global/bin:$PATH\n```\n\n**\"Module not found\"**\n```bash\n# Clear cache and reinstall\ncd react-frontend\nrm -rf node_modules package-lock.json\nnpm install\n```\n\n### API Key Issues\n\n**\"Invalid API key\"**\n- Check .env file format (keys in quotes)\n- Verify keys are active (wait 10 min for OpenWeather)\n- Test keys individually\n\n**\"CORS errors\"**\n- Ensure both backend (5001) and frontend (3000) are running\n- Check firewall settings\n\n### Port Issues\n\n**\"Port already in use\"**\n```bash\n# Find process using port\nnetstat -ano | findstr :5001  # Windows\nlsof -i :5001  # macOS/Linux\n\n# Kill process\ntaskkill /PID <process-id> /F  # Windows\nkill -9 <process-id>  # macOS/Linux\n```\n\n## Platform-Specific Notes\n\n### Windows\n- Use Command Prompt or PowerShell\n- Path separators: `\\` (backslash)\n- Virtual environment: `venv\\Scripts\\activate`\n\n### macOS\n- May need to install Xcode Command Line Tools\n- Use Terminal\n- Virtual environment: `source venv/bin/activate`\n\n### Linux\n- Install python3-venv: `sudo apt install python3-venv`\n- May need build tools: `sudo apt install build-essential`\n- Virtual environment: `source venv/bin/activate`\n\n## Performance Tips\n\n1. **Use SSD storage** for faster npm installs\n2. **Close unnecessary applications** during development\n3. **Use virtual environment** to avoid conflicts\n4. **Keep API keys secure** - never commit to git\n5. **Monitor network usage** - APIs have rate limits\n\n## Next Steps\n\nAfter successful installation:\n\n1. **Read the main README.md** for usage guide\n2. **Try the example orders** in the chat interface\n3. **Explore the agent dashboard** to see system metrics\n4. **Check the tests folder** for additional examples\n5. **Customize the .env file** for your needs\n\n## Getting Help\n\nIf you encounter issues:\n\n1. **Check this troubleshooting section**\n2. **Run the test scripts** in the tests/ folder\n3. **Verify your API keys** are working\n4. **Check system requirements** are met\n5. **Open an issue** with error details\n\n---\n\n**Happy coding! ðŸš€**
